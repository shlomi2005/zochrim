## English

**Zochrim** is a Hebrew Android app for daily mitzvah reminders — tefillin, Omer counting, and halachic zmanim, following the Sephardic (Edot HaMizrach) tradition.

Open source, non-profit, built as a gift to the religious community.

### Features

- 📿 Daily tefillin morning reminder, with morning zmanim for your city
- 🌙 Full 49-day Omer counting with Sephardic nusach
- ☀️ Halachic times for 18 Israeli cities
- 🕯️ Candle lighting reminder before Shabbat / Yom Tov
- 🛡️ Automatically silent on Shabbat, Yom Tov, Chol HaMoed (Sephardic), Tisha B'Av shacharit, Rosh Chodesh handling

### Principles

- **Fully offline** — zero network calls, zero tracking
- **Free forever** — no ads, no IAP
- **Open to all** — MIT licensed, contributions welcome
